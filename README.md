# vc-disableOverlaySoundboard
A plugin for Vencord that disables the in-game overlay soundboard.
<img width="925" height="723" alt="soundboard6" src="https://github.com/user-attachments/assets/dd9b6a75-dfdf-4e5a-926d-067b4719d264" />

Simply disables the entire soundboard while using the in-game overlay, usually accessed by ``Ctrl + ` `` or `Ctrl + /`. For some reason the soundboard in the in-game overlay can not be disabled and the default keybind can not be disabled in the default Discord client.🙃

I made this plugin because I use ``Ctrl + ` `` as my toggle mute keybind and was tired of Discord playing super loud sounds every time I try to mute while in a game.

[Instructions for installation](https://discord.com/channels/1015060230222131221/1257038407503446176)
